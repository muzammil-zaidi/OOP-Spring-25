## How to Use
1. Clone the repository:
   ```sh
   git clone https://github.com/muzammil-zaidi/OOP-Spring-25.git
   ```
2. Navigate to the desired lab folder:
   ```sh
   cd OOP-Spring-25/Lab-01
   ```
3. Compile and run the C++ program:
   ```sh
   g++ task1.cpp -o task1
   ./task1
   ```
